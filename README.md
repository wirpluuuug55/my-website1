# 🌐 Мой первый сайт

Простой сайт, созданный с помощью HTML и CSS.

## 🚀 Возможности

* Заголовок страницы
* Текстовый блок
* Кнопка
* Простое оформление через CSS

## 📂 Структура проекта

```text
project/
├── index.html
└── style.css
```

## 🛠 Используемые технологии

* HTML5
* CSS3

## ▶️ Запуск

1. Скачайте или клонируйте репозиторий.
2. Откройте файл `index.html` в браузере.
